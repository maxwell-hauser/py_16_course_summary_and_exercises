## 3. Represent $(465)_{10}$ in BCD

Group each decimal digit into a 4-bit nibble (BCD):

```
4       6       5
0100    0110    0101
```

**Result:** $(010001100101)_{BCD}$

## 5. Convert BCD $(110010000100)_{BCD}$ to decimal (check validity)

Split into nibbles:

```

```

**Note:** $(1100)_2 = 12$ is not a valid BCD nibble (valid range: 0000–1001). This BCD sequence is **invalid**.

## 6. Two's complement to decimal

Show conversion and final decimal value.

**a) 4-bit:** $(1011)_2$

```
(1011)₂ (4-bit two's complement)
Invert → (0100)₂
Add 1 → (0101)₂ = 5
```

**Result:** $-5$

---

**b) 8-bit:** $(11111001)_2$

```
(11111001)₂ (8-bit two's complement)
Invert → (00000110)₂
Add 1 → (00000111)₂ = 7
```

**Result:** $-7$

---

**c) 8-bit:** $(10011111)_2$

```
(10011111)₂ (8-bit two's complement)
Invert → (01100000)₂
Add 1 → (01100001)₂ = 97
```

**Result:** $-97$

## 7. Subtract unsigned numbers using two's complement
Use two's complement of the subtrahend and add; drop the final carry if present.

a) 11110011 − 11000011

subtrahend: 11000011
two's complement: invert 00111100 + 1 -> 00111101
add: 11110011
     +00111101
     --------
    1 00110000  (discard carry)
     00110000

b) 10001101 − 11111000

subtrahend: 11111000
two's complement: invert 00000111 + 1 -> 00001000
add: 10001101
     +00001000
     --------
     10010101

Result (unsigned): 10010101

## 8. Signed-magnitude -> decimal
